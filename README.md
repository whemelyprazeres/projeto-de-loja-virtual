# Projeto Banco de Dados — Loja Virtual de Photocards

## Descrição do Projeto
Projeto desenvolvido para a disciplina de Banco de Dados utilizando SQL e modelagem relacional.

O sistema simula uma loja virtual de photocards com cadastro de clientes, produtos e pedidos.

---

## Tecnologias Utilizadas

- MySQL
- SQL
- GitHub
- Draw.io

---

## Estrutura do Banco

Tabelas:

- clientes
- photocards
- categorias
- pedidos
- itens_pedido
